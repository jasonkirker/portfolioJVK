Clicky Memory

  • Date :September 21, 2018
  • Skills :HTML, JS, CSS, REACT

This is a memory game where gifs are rendered in random order once a single gif is clicked. The user tries to click all gifs without clicking on any gif twice. The application keeps track of the user's score. The user's score is incremented when clicking a gif for the first time. The score is reset to zero if the user clicks the same gif more than once. The user wins the game if they make it through all the gifs.
